Projetos desenvolvidos

Web Site

Seu Filme

O Seu Filme é uma aplicação web focada na exibição e organização de filmes por categorias, como Ação, Comédia, Ficção e Documentário.

O projeto simula a experiência de uma plataforma de streaming, permitindo que o usuário navegue entre categorias, visualize filmes recomendados e acompanhe a seção Trending com destaque visual em formato moderno (parallax e layout responsivo).

A aplicação utiliza manipulação dinâmica do DOM para atualizar os conteúdos sem recarregar a página, proporcionando uma experiência fluida e interativa.

Tecnologias Utilizadas

  • HTML5: Estrutura semântica da aplicação
  • CSS3: Estilização, responsividade e efeitos visuais (flexbox, grid e parallax)
  • JavaScript (Vanilla JS): Manipulação do DOM, eventos, renderização dinâmica de conteúdo
  • Media Queries:Layout responsivo para desktop e mobile

API RESTful

PontoRecicla-API-REST

API RESTful desenvolvida para cadastro e gerenciamento de usuários e pontos de coleta de materiais recicláveis.

Possui autenticação com JWT, CRUD de locais (restrito ao criador), geração de link do Google Maps via Nominatim e documentação interativa com Swagger.

Tecnologias Utilizadas

  • Node.js, Express, Sequelize, PostgreSQL, JWT, bcryptjs, dotenv, axios, CORS e Swagger (swagger-autogen + swagger-ui-express).

Projeto em React

ReactTimer

Aplicação de temporizador desenvolvida em React com Vite. Permite iniciar, pausar e resetar o tempo, além de registrar um histórico de ciclos concluídos, interrompidos ou em andamento.

Projeto focado em prática de hooks, contexto e organização de estrutura React.

Tecnologias Utilizadas

  • React, Vite, CSS, ESLint e Node.js.
